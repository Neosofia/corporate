# Brand Standards

This page documents Neosofia's brand standards to ensure a consistent and professional presentation across all materials and communications.

## Color Palette

Our brand palette consists of four colors, each serving a specific purpose in our visual identity.

![Neosofia Brand Color Palette](https://github.com/user-attachments/assets/d803aeac-6654-4fd3-a10c-e5d776139dca)

| Color         | Hex       | Usage                                                                                     |
|---------------|-----------|-------------------------------------------------------------------------------------------|
| Golden Yellow | `#FCCD4A` | Icon gradient highlight (top). Use for accents, highlights, and attention-drawing elements. |
| Vibrant Orange | `#F2712C` | Icon gradient midpoint. Use for secondary accents and interactive states (hover, focus).  |
| Deep Red      | `#9B0303` | Icon gradient base and wordmark text. Use for primary text, headings, and key UI elements. |
| Navy Blue     | `#0D4884` | Reserved for contrast and trust. Use for links, informational callouts, and data emphasis. |

### Color Usage Guidelines

- **Deep Red (`#9B0303`)** is the primary brand color. Use it for headings, the wordmark, and primary call-to-action elements.
- **Golden Yellow (`#FCCD4A`)** and **Vibrant Orange (`#F2712C`)** appear in the icon gradient and should only be used as accent colors, never as large background fills.
- **Navy Blue (`#0D4884`)** provides contrast for supporting content such as links, data tables, and informational highlights.
- Do not use brand colors outside of these roles without explicit approval.

## Logo

### Full Logo (Icon + Wordmark)

The full logo combines the circuit-brain icon with the "NEOSOFIA" wordmark. This is the preferred form for most applications.

![Neosofia Full Logo](/website/public/shared/images/Neosofia.png)

### Icon Only

Use the icon-only form when space is limited or when the brand is already established in context (e.g., favicons, app icons, social media profile images).

![Neosofia Icon](https://github.com/user-attachments/assets/56d5fec8-e943-4a98-907f-7c2b85984008)

### Wordmark Only

Use the wordmark-only form when the icon would be too small to render clearly, or in typographic layouts where the icon would create visual imbalance.

![Neosofia Wordmark](https://github.com/user-attachments/assets/429f20ca-ef00-4333-8465-63d42d201d89)

## Logo Spacing

Maintain a clear space around the logo equal to the height of the letter "N" in the wordmark. No text, graphics, or other elements should appear within this exclusion zone.

For the full logo, the gap between the icon and the wordmark should be **equal to the width of the letter "N"** in the wordmark. This spacing must not be reduced.

## Icon Sizing

| Context              | Minimum Size  |
|----------------------|---------------|
| Digital (screen)     | 32 × 32 px    |
| Favicon              | 16 × 16 px    |
| App icon             | 512 × 512 px  |
| Print                | 0.5 in × 0.5 in |

Do not display the icon below 32 × 32 px in digital contexts, as the circuit-board detail becomes illegible. For print, ensure sufficient resolution (minimum 300 DPI).

## Logo Usage Guidelines

- Do not stretch, skew, or distort the logo.
- Do not recolor the wordmark or icon outside of the approved color palette.
- Do not place the logo on busy backgrounds that reduce legibility. Use a white or light neutral background whenever possible.
- Do not add drop shadows, outlines, or other effects to the logo.
- Do not combine the Neosofia logo with other logos or icons in a way that implies a partnership unless one has been formally established.
