# Neosofia Brand Standards

This page defines how to represent the Neosofia brand consistently across products and communications. It covers colors, logo usage, clear space, and sizing.

## Color Palette

Use the Neosofia color palette below. Prefer high-contrast combinations that meet WCAG AA standards.

![Neosofia color palette][palette]

Guidelines:
- Primary brand color: Use for key actions, highlights, and primary navigation.
- Secondary and accent colors: Use sparingly for emphasis, charts, and illustrations.
- Backgrounds: Use dark slate backgrounds from the palette for surfaces; reserve light backgrounds for print artifacts and data tables.
- Accessibility: Maintain a minimum 4.5:1 contrast ratio for body text and 3:1 for large text/icons.

## Logos

Neosofia has two core logo assets: an icon and a wordmark. Use them per the guidance below.

### Icon (mark)

![Neosofia icon][icon]

- Use for favicons, app icons, avatars, and small badges.
- On dark backgrounds use the light/white version; on light backgrounds use the brand-colored or black version.
- Do not apply drop shadows, gradients, outlines, or rotate the icon.

### Wordmark (text-only logo)

![Neosofia wordmark][wordmark]

- Use in headers, footers, and marketing materials where there is sufficient width.
- Keep letter spacing and proportions unchanged.

### Combined lockup (icon + wordmark)

![Neosofia combined lockup][lockup]

- Use when brand recognition is important and there is adequate space.
- Do not re-arrange the order of the icon and wordmark.

## Clear Space and Spacing

To preserve legibility and impact:
- Clear space: Maintain a minimum clear space of "1x" around the logo, where "x" equals the height of the icon. No text, graphics, or UI elements should intrude into this area.
- Icon-to-wordmark spacing (combined lockup): Space the icon and wordmark by "0.5x" (half the icon's width), with a minimum of 16px on web.

## Minimum Sizes

Maintain legibility by adhering to these minimum sizes for digital:
- Icon only: minimum 24px
- Combined lockup: minimum 24px icon height with the wordmark sized proportionally
- Wordmark only: minimum 120px width

Recommended sizes for common placements:
- Navigation/header: 32–48px icon height for the lockup
- Page hero/marketing: 64–96px icon height for the lockup
- Favicons/app icons: provide 16, 32, 64, 128, 256px square assets

## Incorrect Usage

Avoid the following:
- Altering colors outside the approved palette
- Adding effects (glow, gradient, drop shadow) not part of the official assets
- Distorting, rotating, or re-spacing the wordmark letters
- Placing logos on low-contrast or visually noisy backgrounds
